What is this?

I made this because I love physics (though I'm not very good at it). I came across this video and got inspired to recreate it using my coding skills. I started with Pygame, but it didn’t work well, so I switched to HTML and JS.

This is like a sandbox: you can enter the mass of the blocks and see how many times the big and small blocks collide.

The most interesting part is when the big block’s mass is 1, 10, 100, 1000, etc. The number of collisions starts revealing the digits of π, which you can see in the yellow text.
